Pasos para subir un proyecto a github

  1. Crear el repositorio
  2. Abrir una terminal (Es importante que primero hayamos descargado el git bash)
  3. correr git init dentro de la terminal
  4. Se corre git add . para empezar a rasterar los archivos
  5. Se corre el git commit
  6. Se escoge la rama donde se va a guardar
  7. se agrega el origen donde se va a subir el proyecto
  8. Se corre gitPush el cual es el comando para ya subir el programa a github
